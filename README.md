# Aplicación de Gestión de Eventos Comunitarios

Aplicación móvil desarrollada en React Native para gestionar eventos y actividades comunitarias locales. Permite la creación de eventos, confirmación de asistencia (RSVP), interacción social y visualización de estadísticas.

## 🚀 Guía de Instalación y Ejecución (Para Desarrolladores)

Sigue estos pasos para correr el proyecto localmente en tu computadora:

### Requisitos Previos
* Tener instalado [Node.js](https://nodejs.org/)
* Tener cuenta de Expo / Aplicación Expo Go instalada en el celular.

### Pasos de Instalación
1. **Clonar el repositorio:**
   ```bash
   git clone [https://github.com/RoG1-bit/gestion-eventos-app.git]
   cd gestion-eventos-app
2. **Instalar las dependencias:**
   npm install
3.  **Ejecutar la aplicacion:**
   npx expo start
4.  **Opciones de visualizacion:**
    * Presiona w en la terminal para abrirlo en el navegador web.
    * Escanea el código QR con la aplicación Expo Go en tu dispositivo móvil.
    * Presiona a para abrirlo en un emulador de Android.
